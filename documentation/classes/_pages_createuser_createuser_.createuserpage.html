register

  • register(): void
    • Defined in pages/createuser/createuser.ts:57
    name:

    register()

    description:

    Guarda los datos de registro introducidos por el usuario. Los datos se guardarán en local y se cargarán en el servidor en la siguiente sincronización (inicio de la app). Comprueba si hay conexión y datos de usuarios en local. Si hay conexión y no datos locales, los intenta descargar y añade el nuevo registro a esta lista. Si no hubiera conectividad ni datos cargados en local, el usuario se generaría igualmente y se sincronizaría con la plataforma en el siguiente inicio de la app. Hasta que no se realizase una sincronización de los datos, los nuevos usuarios no tendrían ningún rol (owner, register o visualizer), por lo que solo podrían escanear las etiquetas y se les mostrará un string con el escaneo sin ningún otro dato.

    Returns void
